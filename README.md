# テストを実施する場合
このプロジェクトをローカルでテストする場合、各自で Google Cloud Platform (GCP) や Firebase の環境を準備する必要があります。

# テストの実施手順
* 📂 server_functions をセットアップ
* 📂 server　をセットアップ
* 📂 client  をセットアップ
